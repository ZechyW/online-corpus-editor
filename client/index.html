    Use category:<n>, comment:<word>, tag:<tag> and language:<language> to search for specific words within those fields.

    By default, the query will include all the search terms provided - There is no need to explicitly insert '&' or 'and' between terms. To exclude search terms, use the - prefix; it goes after the colon for filters.

    The uppercase OR operator can be used to combine word, phrase, or filter queries; brackets are not supported.

    For example, searching millefuille OR "delicious burger" OR christmascakes comment:-tasty will return records that contain any of the first three terms but do not have the word tasty in their comments.
